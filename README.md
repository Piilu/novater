## Kuidas tööle panna

1. Jooksuta `npm i`
2. Loo `.env` fail `.env.example` põhjal
3. Kui andmebaasi pole siis jooksuta `docker compose up -d`
4. Lõpuks jooksuta `npm run start:migrate:dev`


## Mida teeksin juurde?
- Võimaluse osta ümberistumisega piletit
- Lazy loading
- Parema filtreerimis loogika
